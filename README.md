# 🚀 High-Performance Chrome Extension with React & Vite

A lightweight, secure, and modern Chrome Extension built using **React 18**, **Vite**, and **Manifest V3**. This project was developed for a client on Upwork to provide a seamless user experience within the browser popup.

## ✨ Key Features
- **Modern Tech Stack:** Built with React and Vite for ultra-fast development and performance.
- **Manifest V3:** Fully compliant with the latest Chrome Extension security standards.
- **Clean UI:** Responsive and user-friendly interface.
- **Optimized Build:** Small bundle size for quick loading.

## 🛠️ Tech Stack
- **Frontend:** [React.js](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** CSS3 / Tailwind (qaysi birini ishlatgan bo'lsangiz shuni qoldiring)
- **API:** Chrome Extension API (Manifest V3)
